# Valorant Esports Predictor

**Valorant Esports Predictor** est une application communautaire dédiée aux fans de l’esport VALORANT.  
L’objectif du projet est d’améliorer l’expérience des spectateurs en centralisant les informations liées aux compétitions et en proposant un système de prédictions gratuites et ludiques pour les matchs professionnels.

> $*Projet en cours de développement**

---

## Vision du projet

Valorant Esports Companion vise à devenir une plateforme complète pour suivre l’écosystème esport VALORANT, avec notamment :

- **Calendrier des compétitions** (VCT, Challengers, événements majeurs)
- **Résultats et statistiques des tournois**
- **Fiches équipes et joueurs**
- **Système de prédictions gratuites** sur les matchs à venir :
  - Vainqueur du match  
  - Score exact  
  - MVP du match  
- **Système de points fictifs (gamification)** basé sur la précision des prédictions
- **Classement global des utilisateurs**
- **Classements privés entre amis**

**Aucun pari, aucune monnaie réelle ou virtuelle échangeable, et aucune fonctionnalité payante.**  
Le système est purement récréatif et communautaire.

---

## APIs Riot prévues

Le projet s’appuie (ou prévoit de s’appuyer) sur les APIs Riot suivantes :

- **VALORANT Content API** — métadonnées, équipes, tournois, calendriers
- **VALORANT Esports / Compétitions API** — données officielles des matchs

Ce projet est conçu pour respecter les règles et recommandations de Riot Games :

- ✔ Pas de pari, de gambling ou de monétisation
